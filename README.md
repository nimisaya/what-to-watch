# What to watch?

## Overview

Not sure what to watch? Use **What to watch?** to find the movie or tv show for you. 

## Tech

Android app using Jetpack Compose and TMDB API

## Features

Search The Movie DB for a movie, tv show or creator (actor, director etc.)
